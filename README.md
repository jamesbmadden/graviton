# Graviton
Web Components Designed by James Madden, Built with LitElement, Loosely Following Material Spec.

## Install
```
git clone https://github.com/jamesbmadden/graviton
cd graviton
yarn install
```

## Build
```
yarn build
```

## Use in a Project
```
yarn link
```
Then, in your project's directory, run
```
yarn link graviton
```
